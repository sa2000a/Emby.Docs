The guides will go over the following:

* how to setup regular backups using the Backup & Restore Plugin. See [Backing Up Emby Server](Backup-Using-Plugin.md).
* how to migrate your settings from Emby Server from one machine to another. See [Using Restore to add an Additional Server](Backup-Restore-Add-New-Server.md) and [Using Restore to migrate to a New System](Backup-Restore-To-New-System.md).
* how to fully migrate to a new system. See [Using Restore to migrate to a New System](Backup-Restore-To-New-System.md). 
* how to restore your server to the last backed up state. See [Restoring Last Backup](Backup-Restore-Last-Backup.md).
* how to restore your server to a specific backup date of the databases. See [Restoring an Earlier Backup of the Database](Backup-Restore-Earlier-Backup.md).
* how to copy playback state and favorites from one user account to another. See [Using Restore to Copy Playback States and Favorites](Backup-Restore-Playbacks-And-Favorites.md).

All the above use the Emby Server **Backup & Restore Plugin** and will require [Emby Premiere](Emby-Premiere.md).

An article is also provided for manual backups. See [How to Backup Manually](Backup-Manual-Backups.md).
